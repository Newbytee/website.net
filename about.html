The Kontalk Messaging Project

A new way of communicating.
Who we are We are the founders and developers of the Kontalk network and we currently run the main server (prime.kontalk.net) on our own expenses. Donations are welcome :)
The network The Kontalk network is currently made up of one server (our own), but anyone is free to link to the network.
Why should I join your network? The Kontalk network is a distributed network. It works using different servers but it acts like it's a big giant server. The purpose of the network is to create a unique point of access for every user in the world using the service. Creating a new network will just divide users – users in the same network can see each other.
Web of Trust A server is untrusted at the beginning; by means of a vote system – based on strong asymmetric encryption – and by working actively in the network it can become a trusted server and get more privileges.
It's not a system for keeping control of the network, it's a democratic network based on public vote.
